Network Debugging

Open your browser's developer tools (F12) and check the Network tab for more details about the requests.

Common issues:

  • CORS errors - The API needs to allow requests from your domain
  • Authentication errors - Check if the API key is correct
  • Network errors - Check if the API is accessible from your location
